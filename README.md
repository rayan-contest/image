Rayan World Finals Image Builder
=====================================================

The Contest Image Builder contains a set of bash scripts used to build
the contest image for the [Rayan World Finals](http://rayan.global/) in Tehran, Iran.

Installation
------------

To install the Rayan World Finals contest image, follow the instructions below:

  * install Ubuntu 24.04 LTS Desktop on your system
  * create a user named ``rayan``
  * run ``sudo bash install.sh``
